# Expense Tracker

A modern, minimalistic, and responsive Expense Tracker Web Application built with React.js and Tailwind CSS.

## Features

- **Transaction Management**: Add income and expenses with categories.
- **Dashboard Overview**: summary cards for Total Balance, Income, and Expenses.
- **Visual Analytics**: Interactive charts showing expense distribution and income vs. expenses.
- **Smart Recommendations**: AI-like insights based on your spending habits.
- **CSV Export**: Download your transaction data for external analysis.
- **Local Storage**: Data persists across browser sessions.
- **Responsive Design**: Looks great on desktop and mobile.

## Tech Stack

- React
- Tailwind CSS
- Recharts (for charts)
- Lucide React (for icons)
- Framer Motion (for animations)
- Vite (for fast development)

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the application:
    ```bash
    npm start
    ```
    Or
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```

## Project Structure

- `src/components`: Reusable UI components
- `src/App.jsx`: Main application logic and state management
- `src/index.css`: Global styles and Tailwind directives
